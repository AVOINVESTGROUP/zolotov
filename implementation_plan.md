# План реализации каталога и карточки товара

Необходимо создать полноценный путь покупателя: от просмотра всех товаров до детального изучения конкретного изделия.

## User Review Required

> [!IMPORTANT]
> **Фильтрация:** На первом этапе фильтрация по категориям будет работать через URL-параметры (например, `/catalog?category=rings`), чтобы сохранить производительность и SEO (Server-Side Rendering).
> **Иконки:** В коде будут использоваться `lucide-react`. Пожалуйста, убедитесь, что выполнили `npm install lucide-react`.

## Proposed Changes

### 1. Страница каталога (`/catalog`)

#### [NEW] [page.tsx](file:///c:/Dev/Zolotov/app/catalog/page.tsx)

## Verification Plan

### Automated Tests
- `npm run build` для проверки типов и роутинга.

### Manual Verification
- Переход с главной на страницу каталога.
- Клик по кнопке категории — товары должны отфильтроваться (через `searchParams`).
- Переход в карточку товара — проверка отображения цен и описания.
- Добавление товара в корзину.
