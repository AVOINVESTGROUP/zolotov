# План реализации Корзины и Checkout

Необходимо реализовать завершающий этап воронки продаж: от управления списком покупок до формирования заказа в базе данных WooCommerce.

## User Review Required

> [!IMPORTANT]
> **Приём платежей:** На текущем этапе будет реализован метод "Оплата при доставке" (Cash on Delivery) или "Банковский перевод". Полноценный эквайринг (Stripe/Checkout.com) требует отдельного обсуждения и настройки ключей.
> **Безопасность:** Создание заказа будет происходить через Next.js Route Handler (`/api/checkout`), чтобы скрыть секретные ключи WooCommerce (`WC_CONSUMER_SECRET`) от браузера.
